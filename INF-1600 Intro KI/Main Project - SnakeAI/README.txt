In the src/ directory there is an AI that uses deep Q learning to teach itself to play the classic videogame Snake.

Necessary libraies:
  - pytorch
  - pygame
  - numpy
 
main.py runs the program and uses pytorch to create an agent to play the game and pygame to create the visuals.
  - environment.py is an implemenation of snake that is playable by an agent.
  
 snakeHuman.py is the same program as environment.py but is playable by a human.
